SECURITY PILLARS

Built for finance-grade controls.

Four pillars that finance teams, auditors, and security reviewers expect — enforced across every Orion agent workflow.

✓

SOC 2 Type II Certified

revzio has completed an independent SOC 2 Type II audit. Our controls are verified by a third-party auditor. Full report available under NDA.

Data Encrypted Everywhere

All data encrypted in transit (TLS 1.3) and at rest (AES-256). Encryption keys managed in a dedicated KMS — never stored with your data.

Role-Based Access Control

Granular permissions for preparers, reviewers, and approvers. Orion mirrors your existing control matrix without configuration overhead.

◎

Data Residency

Your financial data is processed and stored in your chosen region. We do not use customer data to train AI models.
